# ellipsis-parser

Парсер документов и веб-страниц для автоматического обнаружения 
глагольного эллипсиса в русскоязычных текстах.

## Запуск
`java -jar ellipsis-parser.jar data/input/razbor.txt`

## Входные данные
- `data/input/razbor.txt` — 100 предложений с глагольным эллипсисом

## Результаты эксперимента
- `output/results.txt` — найденные предложения
- `output/results.csv` — то же в формате CSV
